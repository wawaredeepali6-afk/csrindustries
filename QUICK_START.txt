CSR INDUSTRIES WEBSITE - QUICK START
========================================

Your professional React.js website has been created successfully!

STEP 1: INSTALL DEPENDENCIES
-----------------------------
Double-click: install.bat
OR run in terminal: npm install

STEP 2: START DEVELOPMENT SERVER
---------------------------------
Double-click: start.bat
OR run in terminal: npm start

Website will open at: http://localhost:3000

STEP 3: CUSTOMIZE CONTENT
--------------------------
1. Replace placeholder images with real product photos
2. Update contact information (phone, email, address)
3. Add real product specifications
4. Update company details in About page
5. Add actual project photos to Gallery

FILES TO EDIT:
--------------
Contact Info:
- src/components/Header.js (line 15-16)
- src/components/Footer.js (line 45-48)
- src/pages/Contact.js (line 40-50)

Products:
- src/pages/Products.js (line 8-60)

Company Info:
- src/pages/About.js (all content)
- src/pages/Home.js (hero section)

Gallery:
- src/pages/Gallery.js (line 8-20)

STEP 4: BUILD FOR PRODUCTION
-----------------------------
Double-click: build.bat
OR run in terminal: npm run build

Upload the 'build' folder to your web hosting.
